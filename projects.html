Old Age Home Mobile Application [Elderly Care App]

- Dedicated platform designed to cater to the unique needs of elderly living and providing easy ways for staff and caregivers to take care of the elderly.
- Facilitating a streamlined process for staff and caregivers to document medical records obtained from elderly individuals.
- Developed a pill reminder system to prompt elderly individuals to take their medication, alongside incorporating chat and messaging features to facilitate connections and foster new friendships.
- Delivering live remote monitoring capabilities to enable users to oversee the well-being of their elderly parents in real-time.
- Presenting a line graph depicting the historical health records of elderly individuals, followed by the calculation and display of their overall health percentage through a pie chart.
- Incorporating an emergency help function that allows elderly individuals to send alert messages and make emergency calls when needed.
- Utilizing Firebase for database storage and ZEGOCLOUD for remote monitoring management, particularly for live streaming functionalities.
- Technologies used: Android Studio, Java, Firebase, ZEGOCLOUD, Google Maps API

Fitness Mobile Application

Platform for assisting bodybuilders to record their daily exercise routine and enable communication with other users in the community.
- Workout Tracking: Allows users to log their workout routines, including sets, reps, and weights used for each exercise.
- Exercise Library: Provides a comprehensive list of exercises to help users record their exercies more efficiently.
- Weight Tracking: Enables users to track their body weight over time, providing graphs to visualize progress.
- Measurement Tracking: Users can log their body measurements (e.g., chest, waist, hips) to monitor changes in their physique.
- Functions: Facilitates interaction with other users through posting their daily/weekly/monthly exercise routine.
- Goal Setting: Allows users to set fitness goals and provides reminders and motivational notifications to help achieve them.

PHP Login & Sign Up Module

This web module is designed to provide a secure and efficient login and sign-up system for users using HTML, CSS, PHP, XAMPP, and phpMyAdmin. The key features of this module include:

- User Registration: Allows new users to create an account by providing necessary information such as username, email, and password. The system performs validations to ensure data integrity and security.

- User Login: Enables registered users to log in to the system using their credentials. The login process includes error handling for incorrect usernames or passwords.

- Password Recovery: If a user forgets their password, they can request a password reset. The system sends an email verification link to the user's registered email address. Upon verification, users can reset their passwords securely.

- Database Management: Uses phpMyAdmin in conjunction with XAMPP to manage the database. The database stores user information securely, including hashed passwords for added security.

- Technologies Used: HTML, CSS, PHP, XAMPP, phpMyAdmin
